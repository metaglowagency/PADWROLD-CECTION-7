## Media Assets & Dividers

The application uses high-impact parallax dividers between major logical modules. These are managed via `assets.ts`.

### Updating Dividers
To update images, place high-resolution JPGs in `/public/images/dividers/` and update the corresponding path in `assets.ts`.
