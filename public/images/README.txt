PADWORLD IMAGE ASSETS
=====================

Place your local image files in this folder.

How to use:
1. Save your image here (e.g., 'dashboard-bg.jpg').
2. In 'assets.ts', update the URL to point to the relative path:
   
   ecosystem: {
     url: "/images/dashboard-bg.jpg",
     ...
   }

Note: Files in the 'public' folder are served at the root path '/'.
